Вход в систему:
Ввод логина/пароля: Создание формы с полями для ввода логина и пароля.
Авторизация в ИС: Обработка введенных данных и проверка их наличия в базе данных.

Главное меню:
Переход на страницу "Управление поставками деталей": Кнопка.
Переход на страницу "Планирование выпуска продукции": Кнопка.
Переход на страницу "Управление продажами": Кнопка.
Переход на страницу "Управление отделами": Кнопка.
Переход к настройкам приложения: Кнопка.

Настройки приложения:
Задание темы оформления: Выбор цветовой схемы через выпадающий список или переключатели.
Задание валюты: Выбор валюты из списка или ввод пользовательского значения.
Просмотр политики конфиденциальности: Кнопка.

Страница "Управление поставками деталей":
Переход на страницу "Поставщики", "Контракты на поставку", "Запасы деталей": Кнопки.

Страница "Поставщики":
Просмотр данных в виде таблицы, поиск, сортировка: Использование табличных компонентов и фильтров.
Добавление, редактирование, удаление информации: Формы для ввода, редактирования данных и кнопки для удаления.

Форма добавления/редактирования информации о поставщике:
Ввод/изменение полей: Формы с полями для ввода/изменения данных.
Сохранение информации в ИС: Кнопка для сохранения данных в базе.

Страница "Контракты на поставку" и "Запасы деталей":
Аналогично "Поставщикам": Использование табличных компонентов, фильтров и форм для взаимодействия с данными.

Страница "Планирование выпуска продукции":
Переход на страницы "Текущий план", "Следующий план", "Номенклатура продукции": Кнопки.

Страницы "Текущий план", "Следующий план", "Номенклатура продукции":
Аналогично разделу "Поставщики": Использование табличных компонентов, фильтров и форм для взаимодействия с данными.

Утверждение плана:
Выбор пунктов плана: Флажки или другие элементы для выбора требуемых пунктов.
Утверждение и перенос в текущий план: Кнопка для выполнения действия.

Страница "Управление продажами":
Переход на страницы "Заказчики", "Заказы", "Статистика": Кнопки.

Страницы "Заказчики" и "Заказы":
Аналогично разделам "Поставщики" и "Контракты на поставку": Табличные компоненты, фильтры и формы для взаимодействия с данными.

Страница "Статистика":
Просмотр статистики: Использование графиков, диаграмм или таблиц для отображения данных.

Страница "Управление отделами":
Переход ко внешней системе аутентификации: Кнопка.
Переход ко внешней системе бухгалтерского учета: Кнопка
